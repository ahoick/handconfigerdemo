# 对焦模组控制系统

基于 STM32 + MS41929 的双轴步进电机对焦控制系统，支持**手势识别**和**语音控制**。

## 硬件架构

```
┌──────────────────────────────────────────────────┐
│                    STM32F103C8T6                   │
│                                                    │
│  USART1 (PA9/PA10) ←──→ PC 上位机 (手势识别)        │
│      115200bps, 单字节指令协议                      │
│                                                    │
│  USART3 (PB10/PB11) ←── LD3320 语音模块            │
│      9600bps, AA 55 [ID] 55 AA 帧协议              │
│                                                    │
│  SPI1 (PA4~PA7) ──→ MS41929 双通道电机驱动         │
│                                                    │
│  PB1 ──→ 七彩自动闪烁 LED                          │
│  PB5 ──→ IN1 (直流电机)                            │
│  PB6 ──→ IN2 (直流电机)                            │
└──────────────────────────────────────────────────┘
```

## 串口通信协议（PC ↔ STM32）

**单字节指令 + 单字节响应**，115200bps，8N1。

### 电机控制

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x00` | 心跳包（刷新看门狗） | `0x55` |
| `0x01` | AB电机 正转 | `0x55` |
| `0x02` | AB电机 反转 | `0x55` |
| `0x03` | AB电机 停止 | `0x55` |
| `0x04` | AB电机 刹车 | `0x55` |
| `0x05` | CD电机 正转 | `0x55` |
| `0x06` | CD电机 反转 | `0x55` |
| `0x07` | CD电机 停止 | `0x55` |
| `0x08` | CD电机 刹车 | `0x55` |

### 外设控制

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x09` | 七彩LED 开 (PB1=HIGH) | `0x55` |
| `0x0A` | 七彩LED 关 (PB1=LOW) | `0x55` |
| `0x0C` | AB+CD 同时急停 | `0x55` |

### 速度设置

| 指令范围 | 功能 | 响应 |
|----------|------|------|
| `0x10` ~ `0x73` | 速度 1~100（指令 = 速度 + 0x0F） | `0x55` |

### 心跳机制

- 上位机每 **100ms** 发送 `0x00` 心跳包
- 下位机 **1 秒**无有效指令 → 自动刹车所有电机 + 上报 `0x99`

## 手势识别

### 8 种手势

| 手势 | 动作 |
|------|------|
| ☝️ `1` | AB 正转 |
| ✌️ `2` | AB 反转 |
| 🤟 `3` | CD 正转 |
| 🖖 `4` | CD 反转 |
| 🖐️ `5` | 七彩灯 **开** |
| 👌 `OK` | 七彩灯 **关** |
| 👍 `Good` | **急停** |
| ✊ `Fist` | **急停** |

- 防抖机制：手势需稳定保持 **0.3 秒** 才触发指令
- 技术栈：MediaPipe Hand Landmark → 32维特征 → KNN(k=3) 分类

## 语音控制（LD3320）

### 7 条语音指令

| 关键词 ID | 示例指令 | 动作 |
|-----------|----------|------|
| 1 | "对焦正转" | AB 正转 |
| 2 | "对焦反转" | AB 反转 |
| 3 | "变焦正转" | CD 正转 |
| 4 | "变焦反转" | CD 反转 |
| 5 | "停止" | AB+CD 急停 |
| 6 | "开灯" | 七彩LED 亮 |
| 7 | "关灯" | 七彩LED 灭 |
| 8 | "播放音乐" | 播放音乐 |

> LD3320 → STM32 通信格式：`AA 55 [ID] 55 AA`

## 运行上位机

```bash
# 依赖
pip install opencv-python pyserial pyqt5 numpy mediapipe scikit-learn onnxruntime-gpu

# 启动
python hand-main/hand-main/creatation/motor_control_app/main.py
```

### 操作流程

1. 选择串口 → 点击「连接串口」
2. 选择摄像头 → 点击「打开摄像头」
3. 勾选「手势识别」→ 做出手势控制电机/LED
4. 或使用 LD3320 语音模块说出指令
