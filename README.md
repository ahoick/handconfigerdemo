# 对焦模组控制系统

基于 STM32 + MS41929 的双轴步进电机对焦控制系统，支持**手势识别**和**语音控制**。

## 硬件架构

### 主控芯片：STM32F103C8T6（72MHz，64KB Flash，20KB SRAM）

### 完整引脚分配

| 引脚 | 功能 | 方向 | 连接目标 | 说明 |
|------|------|------|----------|------|
| **USART1 — 上位机通信（115200bps，8N1）** |||||
| PA9 | USART1_TX | 输出 → | USB转TTL模块 RXD | STM32 发送指令响应到 PC |
| PA10 | USART1_RX | 输入 ← | USB转TTL模块 TXD | STM32 接收 PC 手势/控制指令 |
| **USART3 — LD3320 语音模块（9600bps，8N1）** |||||
| PB10 | USART3_TX | 输出 → | LD3320 RXD | STM32 发送配置指令到语音模块 |
| PB11 | USART3_RX | 输入 ← | LD3320 TXD | STM32 接收语音识别结果帧 |
| **SPI1 — MS41929 电机驱动** |||||
| PA4 | CS (NSS) | 输出 → | MS41929 CS | SPI 片选信号 |
| PA5 | SCK | 输出 → | MS41929 SCK | SPI 时钟信号 |
| PA6 | MISO | 输入 ← | MS41929 SOUT | SPI 主机输入 |
| PA7 | MOSI | 输出 → | MS41929 SIN | SPI 主机输出 |
| **控制信号** |||||
| PA2 | VDFZ | 输出 → | MS41929 VD_FZ | 同步触发信号（50Hz脉冲） |
| PA3 | RSTB | 输出 → | MS41929 RSTB | 硬件复位（低电平复位） |
| **GPIO 外设** |||||
| PB1 | GPIO_OUT | 输出 → | 七彩自动闪烁LED VCC | 高电平=模块通电闪烁，低电平=灭 |
| PB5 | GPIO_OUT | 输出 → | IR-CUT IN1 | 直流电机控制引脚1 |
| PB6 | GPIO_OUT | 输出 → | IR-CUT IN2 | 直流电机控制引脚2 |

## 串口通信协议（PC ↔ STM32）

**单字节指令 + 单字节响应**，115200bps，8N1。

### 电机控制

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x00` | 心跳包（刷新看门狗） | `0x55` |
| `0x01` | AB电机 正转 | `0x55` |
| `0x02` | AB电机 反转 | `0x55` |
| `0x03` | AB电机 停止 | `0x55` |
| `0x04` | AB电机 刹车 | `0x55` |
| `0x05` | CD电机 正转 | `0x55` |
| `0x06` | CD电机 反转 | `0x55` |
| `0x07` | CD电机 停止 | `0x55` |
| `0x08` | CD电机 刹车 | `0x55` |

### 外设控制

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x09` | 七彩LED 开 (PB1=HIGH) | `0x55` |
| `0x0A` | 七彩LED 关 (PB1=LOW) | `0x55` |
| `0x0C` | AB+CD 同时急停 | `0x55` |

### 速度设置

| 指令范围 | 功能 | 响应 |
|----------|------|------|
| `0x10` ~ `0x73` | 速度 1~100（指令 = 速度 + 0x0F） | `0x55` |

### 心跳机制

- 上位机每 **100ms** 发送 `0x00` 心跳包
- 下位机 **1 秒**无有效指令 → 自动刹车所有电机 + 上报 `0x99`

## 手势识别

### 8 种手势

| 手势 | Label | 动作 | 对应串口指令 |
|------|-------|------|-------------|
| ☝️ 1指 | `1` | AB 正转 | `0x01` |
| ✌️ 2指 | `2` | AB 反转 | `0x02` |
| 🤟 3指 | `3` | CD 正转 | `0x05` |
| 🖖 4指 | `4` | CD 反转 | `0x06` |
| 🖐️ 5指 | `5` | 七彩灯 **开** | `0x09` |
| 👌 OK | `OK` | 七彩灯 **关** | `0x0A` |
| 👍 点赞 | `Good` | **急停** | `0x0C` |
| ✊ 握拳 | `Fist` | **急停** | `0x0C` |

- 防抖机制：手势需稳定保持 **0.3 秒** 才触发指令
- 技术栈：MediaPipe Hand Landmark → 32维特征 → KNN(k=3) 分类

## 语音控制（LD3320）

### 8 条语音指令（7条串口输出 + 1条本地播放）

| 关键词 ID | 语音指令 | 串口发送帧 | STM32 动作 |
|-----------|----------|-----------|------------|
| 1 | "对焦正转" | `AA 55 01 55 AA` | AB 正转 |
| 2 | "对焦反转" | `AA 55 02 55 AA` | AB 反转 |
| 3 | "变焦正转" | `AA 55 03 55 AA` | CD 正转 |
| 4 | "变焦反转" | `AA 55 04 55 AA` | CD 反转 |
| 5 | "停止" | `AA 55 05 55 AA` | AB+CD 急停 |
| 6 | "开灯" | `AA 55 06 55 AA` | 七彩LED 开 (PB1=HIGH) |
| 7 | "关灯" | `AA 55 07 55 AA` | 七彩LED 关 (PB1=LOW) |
| 8 | "播放音乐" | —（不发送） | LD3320 本地播放音乐 |

> LD3320 → STM32 通信格式：`AA 55 [ID] 55 AA`，9600bps，USART3 (PB10/PB11)。ID 8 由 LD3320 本地处理，不输出串口数据。

## 运行上位机

```bash
# 依赖
pip install opencv-python pyserial pyqt5 numpy mediapipe scikit-learn onnxruntime-gpu

# 启动
python hand-main/hand-main/creatation/motor_control_app/main.py
```

### 操作流程

1. 选择串口 → 点击「连接串口」
2. 选择摄像头 → 点击「打开摄像头」
3. 勾选「手势识别」→ 做出手势控制电机/LED
4. 或使用 LD3320 语音模块说出指令
