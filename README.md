# 对焦模组控制系统

基于 STM32 + MS41929 的双轴步进电机对焦控制系统，支持**手势识别**和**语音控制**。

## 硬件架构

### 主控芯片：STM32F103C8T6（72MHz，64KB Flash，20KB SRAM）

### 完整引脚分配

| 引脚 | 功能 | 方向 | 连接目标 | 说明 |
|------|------|------|----------|------|
| **USART1 — 上位机通信（115200bps，8N1）** |||||
| PA9 | USART1_TX | 输出 → | USB转TTL模块 RXD | STM32 发送指令响应到 PC |
| PA10 | USART1_RX | 输入 ← | USB转TTL模块 TXD | STM32 接收 PC 手势/控制指令 |
| **SPI1 — MS41929 电机驱动** |||||
| PA4 | CS (NSS) | 输出 → | MS41929 CS | SPI 片选信号 |
| PA5 | SCK | 输出 → | MS41929 SCK | SPI 时钟信号 |
| PA6 | MISO | 输入 ← | MS41929 SOUT | SPI 主机输入 |
| PA7 | MOSI | 输出 → | MS41929 SIN | SPI 主机输出 |
| **控制信号** |||||
| PA2 | VDFZ | 输出 → | MS41929 VD_FZ | 同步触发信号（50Hz脉冲） |
| PA3 | RSTB | 输出 → | MS41929 RSTB | 硬件复位（低电平复位） |
| **GPIO 外设** |||||
| PB0 | GPIO_OUT | 输出 → | 扬声器/蜂鸣器 | 方波驱动（需外接三极管），各指令不同音调 |
| PB1 | GPIO_OUT | 输出 → | 七彩自动闪烁LED VCC | 高电平=模块通电闪烁，低电平=灭 |
| PB5 | GPIO_OUT | 输出 → | IR-CUT IN1 | 直流电机控制引脚1 |
| PB6 | GPIO_OUT | 输出 → | IR-CUT IN2 | 直流电机控制引脚2 |

> **接线要点**：TX 接对方 RX，RX 接对方 TX，GND 共地。

### 串口接线明细（极易接反，务必核对）

```
┌─────────────────────┬──────────────────────┬──────────────────────────┐
│      连接路径        │     发送端 → 接收端    │         说明              │
├─────────────────────┼──────────────────────┼──────────────────────────┤
│  PC ↔ STM32 (USART1)│ PC(USB-TTL TXD) → PA10│ STM32 接收上位机指令      │
│                     │ PC(USB-TTL RXD) ← PA9 │ STM32 向上位机回复 0x55   │
│                     │ PC(USB-TTL GND) — GND │ 共地，否则通信不稳定       │
├─────────────────────┼──────────────────────┼──────────────────────────┤
│  MS41929 ↔ STM32    │ PA4  → MS41929 CS    │ SPI 片选（软 NSS）         │
│  (SPI1)             │ PA5  → MS41929 SCK   │ SPI 时钟                  │
│                     │ PA6  ← MS41929 SOUT  │ SPI 主机输入              │
│                     │ PA7  → MS41929 SIN   │ SPI 主机输出              │
│                     │ PA2  → MS41929 VD_FZ │ 同步触发 (50Hz)           │
│                     │ PA3  → MS41929 RSTB  │ 硬件复位 (低有效)         │
└─────────────────────┴──────────────────────┴──────────────────────────┘
```

> **排查口诀**：没有 0x55 响应 → 检查 USART1 TX/RX 是否接反；电机不转 → 检查 SPI 四线 + VD_FZ + RSTB。

## 串口通信协议（PC ↔ STM32）

**单字节指令 + 单字节响应**，115200bps，8N1。

### 电机控制

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x00` | 心跳包（刷新看门狗） | `0x55` |
| `0x01` | AB电机 正转 | `0x55` |
| `0x02` | AB电机 反转 | `0x55` |
| `0x03` | AB电机 停止 | `0x55` |
| `0x04` | AB电机 刹车 | `0x55` |
| `0x05` | CD电机 正转 | `0x55` |
| `0x06` | CD电机 反转 | `0x55` |
| `0x07` | CD电机 停止 | `0x55` |
| `0x08` | CD电机 刹车 | `0x55` |

### 外设控制

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x09` | 七彩LED 开 (PB1=HIGH) | `0x55` |
| `0x0A` | 七彩LED 关 (PB1=LOW) | `0x55` |
| `0x0B` | 扬声器（保留，未使用） | `0xEE` |
| `0x0C` | AB+CD 同时急停 | `0x55` |

### 速度设置

| 指令范围 | 功能 | 响应 |
|----------|------|------|
| `0x10` ~ `0x73` | 速度 1~100（指令 = 速度 + 0x0F） | `0x55` |

### 心跳机制

- 上位机每 **100ms** 发送 `0x00` 心跳包
- 下位机 **1 秒**无有效指令 → 自动刹车所有电机 + 上报 `0x99`

## 手势识别

### 8 种手势

| 手势 | Label | 动作 | 对应串口指令 |
|------|-------|------|-------------|
| ☝️ 1指 | `1` | AB 正转 | `速度指令` + `0x01` |
| ✌️ 2指 | `2` | AB 反转 | `速度指令` + `0x02` |
| 🤟 3指 | `3` | CD 正转 | `速度指令` + `0x05` |
| 🖖 4指 | `4` | CD 反转 | `速度指令` + `0x06` |
| 🖐️ 5指 | `5` | 七彩灯 **开** | `0x09` |
| 👌 OK | `OK` | 七彩灯 **关** | `0x0A` |
| 👍 点赞 | `Good` | **急停** | `0x0C` |
| ✊ 握拳 | `Fist` | **急停** | `0x0C` |

> **手势 1~4（电机类）**：与手动点击按钮完全相同的控制路径 → 先发速度设置指令（如 `0x41` 对应速度 50），再发对应的电机动作指令。速度值取界面滑块当前值。

- 防抖机制：手势需稳定保持 **0.3 秒** 才触发指令
- 技术栈：MediaPipe Hand Landmark → 32维特征 → KNN(k=3) 分类

## 扬声器声音反馈

下位机通过 PB0 驱动扬声器，每次收到串口指令会发出不同音调的声音反馈：

| 指令 | 操作 | 声音反馈 |
|------|------|---------|
| `0x01` | AB 正转 | 500Hz / 150ms |
| `0x02` | AB 反转 | 400Hz / 150ms |
| `0x03` | AB 停止 | 300Hz / 80ms |
| `0x04` | AB 刹车 | 200Hz / 100ms |
| `0x05` | CD 正转 | 600Hz / 150ms |
| `0x06` | CD 反转 | 550Hz / 150ms |
| `0x07` | CD 停止 | 350Hz / 80ms |
| `0x08` | CD 刹车 | 250Hz / 100ms |
| `0x09` | 七彩LED 开 | 1000Hz / 60ms（短促） |
| `0x0A` | 七彩LED 关 | 900Hz / 40ms（更短） |
| `0x0C` | 急停 | 200Hz / 300ms（低沉长音） |
| `0x10~0x73` | 速度设置 | 800Hz / 50ms |

> **扬声器接线**：STM32 PB0 → 1kΩ 电阻 → NPN 三极管基极(S8050) → 集电极接扬声器负极，扬声器正极接 5V，发射极接 GND

## 运行上位机

```bash
# 依赖
pip install opencv-python pyserial pyqt5 numpy mediapipe scikit-learn onnxruntime-gpu

# 启动
python hand-main/hand-main/creatation/motor_control_app/main.py
```

### 操作流程

1. 选择串口 → 点击「连接串口」
2. 选择摄像头 → 点击「打开摄像头」
3. 勾选「手势识别」→ 做出手势控制电机/LED
