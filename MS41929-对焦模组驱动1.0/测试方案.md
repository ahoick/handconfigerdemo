### 4.3 测试脚本
```python
# 上位机测试脚本示例（单字节协议）
import serial
import time

CMD_HEARTBEAT  = 0x00
CMD_AB_FORWARD = 0x01
CMD_AB_REVERSE = 0x02
CMD_AB_STOP    = 0x03
CMD_AB_BRAKE   = 0x04
CMD_CD_FORWARD = 0x05
CMD_CD_REVERSE = 0x06
CMD_CD_STOP    = 0x07
CMD_CD_BRAKE   = 0x08
ACK_OK = 0x55

def speed_cmd(speed):
    return speed + 0x0F

ser = serial.Serial('COM3', 115200, timeout=1)

def send_cmd(cmd_byte, desc=""):
    ser.write(bytes([cmd_byte]))
    resp = ser.read(1)
    if len(resp) == 1:
        print(f"{desc}: 发送 0x{cmd_byte:02X} → 响应 0x{resp[0]:02X} {'✓' if resp[0] == ACK_OK else '✗'}")
        return resp[0]
    return None

# 测试 AB 正转
print("测试 AB 正转...")
send_cmd(speed_cmd(100), "设置速度100")
send_cmd(CMD_AB_FORWARD, "AB正转")

# 测试 AB 停止
print("测试 AB 停止...")
send_cmd(CMD_AB_STOP, "AB停止")

# 测试速度设置
print("测试速度设置...")
send_cmd(speed_cmd(50), "设置速度50")

ser.close()
```

## 7. 附录

### 7.1 指令格式参考

单字节指令 + 单字节响应，波特率 115200，8N1。

| 指令 | 值 | 功能 |
|------|----|------|
| HEARTBEAT | `0x00` | 心跳包 |
| AB_FORWARD | `0x01` | AB电机正转 |
| AB_REVERSE | `0x02` | AB电机反转 |
| AB_STOP | `0x03` | AB电机停止 |
| AB_BRAKE | `0x04` | AB电机刹车 |
| CD_FORWARD | `0x05` | CD电机正转 |
| CD_REVERSE | `0x06` | CD电机反转 |
| CD_STOP | `0x07` | CD电机停止 |
| CD_BRAKE | `0x08` | CD电机刹车 |
| 速度 1~100 | `0x10`~`0x73` | 指令 = 速度 + 0x0F |

### 7.2 响应码参考
| 响应码 | 含义 |
|-------|------|
| `0x55` | 执行成功 |
| `0xEE` | 无效指令 |
| `0x99` | 看门狗超时报警 |

### 7.3 寄存器状态参考
通过 SPI 读取 MS41929 寄存器获取电机状态 (0x24=AB, 0x29=CD)：

| 位位置 | 含义 |
|-------|------|
| D13-D12 | 微步进模式（0=64, 1=128, 2=256细分） |
| D11 | LED 状态 |
| D10 | 电机使能状态 |
| D9 | 刹车状态 |
| D8 | 转动方向（1=正转, 0=反转） |
| D7-D0 | 步数设定值 |
