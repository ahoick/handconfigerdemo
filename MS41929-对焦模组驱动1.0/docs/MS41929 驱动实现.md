#### 模块 1：SPI 通信底层函数（寄存器读写）

MS41929 通过**24 位 SPI 数据帧**读写寄存器（说明书 1.2 节数据格式），数据帧结构为：

`[A5~A0(6位地址) + C0(1位读写) + C1(1位保留) + D15~D0(16位数据)]`



* C0=0：写模式；C0=1：读模式；C1 固定为 0。
