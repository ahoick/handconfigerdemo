# MS41929 对焦模组 串口指令协议

## 1. 串口参数

| 参数 | 值 |
|------|----|
| 波特率 | 115200 |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | 无 |
| 超时 | 1000 ms |

---

## 2. 协议概述

采用**单字节指令 + 单字节响应**的简约协议。无帧头、帧尾、校验字节，通信效率高。

- 上位机 → 下位机：1 字节指令
- 下位机 → 上位机：1 字节响应

---

## 3. 指令集总览

### 3.1 心跳保活

| 指令 | 值 | 说明 |
|------|----|------|
| `CMD_HEARTBEAT` | `0x00` | 刷新看门狗计时器，维持连接 |

### 3.2 AB 电机控制（通道 0 / α电机）

| 指令 | 值 | 说明 |
|------|----|------|
| `CMD_AB_FORWARD` | `0x01` | AB 电机正转 |
| `CMD_AB_REVERSE` | `0x02` | AB 电机反转 |
| `CMD_AB_STOP`    | `0x03` | AB 电机自由停止（PSUM=0） |
| `CMD_AB_BRAKE`   | `0x04` | AB 电机刹车抱死（BRAKE=1） |

### 3.3 CD 电机控制（通道 1 / β电机）

| 指令 | 值 | 说明 |
|------|----|------|
| `CMD_CD_FORWARD` | `0x05` | CD 电机正转 |
| `CMD_CD_REVERSE` | `0x06` | CD 电机反转 |
| `CMD_CD_STOP`    | `0x07` | CD 电机自由停止（PSUM=0） |
| `CMD_CD_BRAKE`   | `0x08` | CD 电机刹车抱死（BRAKE=1） |

### 3.4 速度设置

| 指令范围 | 说明 |
|----------|------|
| `0x10` ~ `0x73` | 速度值 1 ~ 100 |

```
指令字节 = 速度值 + 0x0F
速度值   = 指令字节 - 0x0F
```

---

## 4. 响应码

| 响应 | 值 | 含义 |
|------|----|------|
| `ACK_OK`  | `0x55` | 执行成功 |
| `ACK_ERR` | `0xEE` | 无效指令 |
| 超时报警 | `0x99` | 看门狗超时，下位机主动上报（电机已强制刹车） |

---

## 5. 通信流程

### 5.1 电机控制

```
上位机                                下位机
  |                                      |
  |-------- 速度指令 (0x10~0x73) ------->|
  |<------- 0x55 (ACK_OK) --------------|
  |-------- 电机指令 (0x01~0x08) ------->|
  |<------- 0x55 (ACK_OK) --------------|
  |        (电机开始运转)                |
  |                                      |
  |-------- 心跳 0x00 (每100ms) -------->|  ← 维持看门狗
  |<------- 0x55 -----------------------|
  |                                      |
  |-------- 停止/刹车指令 -------------->|
  |<------- 0x55 -----------------------|
```

### 5.2 心跳保活机制

- 间隔：100ms
- 超时：1000ms（1秒内无任何有效指令 → 下位机自动刹车 + 发送 `0x99`）
- 下位机收到 `0x00` ~ `0x08` 或 `0x10` ~ `0x73` 均刷新看门狗

| 操作 | 发送 (Hex) | 响应 (Hex) |
|------|-----------|-----------|
| 心跳 | `00` | `55` |
| AB 正转 | `01` | `55` |
| AB 反转 | `02` | `55` |
| AB 停止 | `03` | `55` |
| AB 刹车 | `04` | `55` |
| CD 正转 | `05` | `55` |

| CD 停止 | `07` | `55` |

## 7. Python 示例

```python
import serial

# 指令定义
CMD_HEARTBEAT   = 0x00
CMD_AB_FORWARD  = 0x01
CMD_AB_REVERSE  = 0x02
CMD_AB_STOP     = 0x03
CMD_AB_BRAKE    = 0x04
CMD_CD_FORWARD  = 0x05
CMD_CD_REVERSE  = 0x06
CMD_CD_STOP     = 0x07
CMD_CD_BRAKE    = 0x08
ACK_OK = 0x55

def speed_cmd(speed: int) -> int:
    """速度值(1~100) → 指令字节(0x10~0x73)"""
    return speed + 0x0F

ser = serial.Serial('COM3', 115200, timeout=1)

# 设置速度 50
ser.write(bytes([speed_cmd(50)]))
assert ser.read(1)[0] == ACK_OK

# AB 正转
ser.write(bytes([CMD_AB_FORWARD]))
assert ser.read(1)[0] == ACK_OK

# 心跳保活
ser.write(bytes([CMD_HEARTBEAT]))
assert ser.read(1)[0] == ACK_OK

ser.close()
```
