# 下位机 串口指令协议

## 1. 串口参数

| 参数 | 值 |
|------|----|
| 波特率 | 115200 |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | 无 |

---

## 2. 协议说明

**单字节指令协议**，无需帧头/帧尾/校验字节。上位机发送 1 字节指令，下位机回复 1 字节响应。

---

## 3. 指令集

### 3.1 心跳包

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x00` | 心跳包，刷新看门狗计时 | `0x55` |

> 上位机应每 100ms 发送一次心跳包，防止下位机看门狗超时（超时时间 1 秒）。

---

### 3.2 AB 电机控制（通道 0）

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x01` | AB 电机正转 | `0x55` |
| `0x02` | AB 电机反转 | `0x55` |
| `0x03` | AB 电机自由停止（滑行） | `0x55` |
| `0x04` | AB 电机刹车抱死 | `0x55` |

### 3.3 CD 电机控制（通道 1）

| 指令 | 功能 | 响应 |
|------|------|------|
| `0x05` | CD 电机正转 | `0x55` |
| `0x06` | CD 电机反转 | `0x55` |
| `0x07` | CD 电机自由停止（滑行） | `0x55` |
| `0x08` | CD 电机刹车抱死 | `0x55` |
| `0x09` | 七彩LED 开 (PB1 高电平，模块自动闪烁) | `0x55` |
| `0x0A` | 七彩LED 关 (PB1 低电平) | `0x55` |
| `0x0C` | AB + CD 同时急停刹车 | `0x55` |

---

---

### 3.5 速度设置

| 指令范围 | 功能 | 响应 |
|----------|------|------|
| `0x10` ~ `0x73` | 设置速度 | `0x55` |

**换算公式：**
```
指令字节 = 速度值 + 0x0F
速度值 = 指令字节 - 0x0F
```

| 速度值 | 指令字节 |
|--------|----------|
| 1 (最慢) | `0x10` |
| 50 | `0x41` |
| 100 (最快) | `0x73` |

> **注意**：速度设置仅改变速度，不改变电机运行方向。下位机收到后会更新全局速度变量，后续电机运转使用新速度。

---

## 4. 响应码

| 响应 | 含义 |
|------|------|
| `0x55` | 执行成功 |
| `0xEE` | 无效指令 |
| `0x99` | 看门狗超时报警（下位机主动上报） |

---

## 5. 通信流程

### 5.1 电机控制流程

```
1. 上位机发送速度设置指令 (0x10 ~ 0x73)
2. 下位机响应 0x55
3. 上位机发送电机控制指令 (0x01 ~ 0x08)
4. 下位机响应 0x55，电机开始运转
5. 上位机持续发送心跳包 0x00 维持看门狗
```

### 5.2 心跳保活

- 心跳间隔：100ms
- 心跳指令：`0x00`
- 如果超过 1 秒未收到任何有效指令，下位机自动刹车并发送 `0x99` 报警
