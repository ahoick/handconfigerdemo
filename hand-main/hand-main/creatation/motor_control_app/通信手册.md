# 串口通信协议手册

## 1. 概述

本文档描述了上位机与下位机之间的串口通信协议。通信采用RS232/RS485接口，波特率115200，数据位8位，停止位1位，无校验。

## 2. 数据格式

### 2.1 命令帧格式

| 字段 | 长度 | 说明 |
|------|------|------|
| 指令字节 | 1字节 | 命令类型 |
| 参数(可选) | 0-1字节 | 命令参数 |
| 校验和 | 1字节 | 累加和校验(可选) |

### 2.2 响应帧格式

| 字段 | 长度 | 说明 |
|------|------|------|
| 响应码 | 1字节 | 0x55表示成功，0xAA表示失败 |
| 数据(可选) | N字节 | 响应数据 |

## 3. 指令集

### 3.1 电机控制指令

#### 3.1.1 AB电机控制

| 指令 | 功能 | 说明 |
|------|------|------|
| 0x01 | AB电机正转 | 电机顺时针方向旋转 |
| 0x02 | AB电机反转 | 电机逆时针方向旋转 |
| 0x03 | AB电机停止 | 电机自由停止(惯性滑行) |
| 0x04 | AB电机刹车 | 电机立即抱死 |

#### 3.1.2 CD电机控制

| 指令 | 功能 | 说明 |
|------|------|------|
| 0x05 | CD电机正转 | 电机顺时针方向旋转 |
| 0x06 | CD电机反转 | 电机逆时针方向旋转 |
| 0x07 | CD电机停止 | 电机自由停止(惯性滑行) |
| 0x08 | CD电机刹车 | 电机立即抱死 |

### 3.2 速度设置指令

| 指令范围 | 功能 | 说明 |
|----------|------|------|
| 0x10-0x73 | 设置速度 | 速度值范围1-100，对应指令0x10(最慢)至0x73(最快) |

**速度计算公式：**
```
指令字节 = 0x10 + speed - 1
速度值 = 指令字节 - 0x10 + 1
```

**示例：**
- 速度1 → 指令0x10
- 速度50 → 指令0x41 (0x10 + 50 - 1 = 0x41)
- 速度100 → 指令0x73

### 3.3 系统指令

| 指令 | 功能 | 说明 |
|------|------|------|
| 0x00 | 心跳包 | 刷新看门狗，维持连接 |
| 0xAA | 状态查询 | 查询下位机状态 |

## 4. 通信流程

### 4.1 电机控制流程

```
1. 上位机发送速度设置指令(0x10-0x73)
2. 上位机等待响应(0x55)
3. 上位机发送电机控制指令(0x01-0x08)
4. 上位机等待响应(0x55)
5. 电机执行动作
```

### 4.2 位置控制流程

```
1. 上位机发送速度设置指令
2. 上位机发送电机控制指令(正转/反转)
3. 上位机定时查询位置(由心跳定时器触发)
4. 到达目标位置后，上位机发送停止/刹车指令
```

### 4.3 心跳保活机制

- 心跳间隔：100ms
- 心跳指令：0x00
- 作用：维持看门狗计时器，防止下位机复位

## 5. 错误处理

### 5.1 响应码

| 响应码 | 含义 | 处理建议 |
|--------|------|----------|
| 0x55 | 执行成功 | 继续下一步操作 |
| 0xAA | 执行失败 | 检查指令参数，重新发送 |

### 5.2 超时处理

- 命令发送超时：100ms
- 位置更新超时：10s
- 心跳超时：500ms

## 6. 示例代码

### 6.1 Python示例

```python
import serial

class MotorController:
    def __init__(self, port, baudrate=115200):
        self.serial = serial.Serial(port, baudrate, timeout=1)

    def set_speed(self, speed):
        """设置速度 (1-100)"""
        cmd = 0x10 + speed - 1  # 转换为指令字节
        self.serial.write(bytes([cmd]))
        response = self.serial.read(1)
        return response == b'\x55'

    def motor_forward(self):
        """AB电机正转"""
        self.serial.write(bytes([0x01]))
        response = self.serial.read(1)
        return response == b'\x55'

    def motor_reverse(self):
        """AB电机反转"""
        self.serial.write(bytes([0x02]))
        response = self.serial.read(1)
        return response == b'\x55'

    def motor_stop(self):
        """AB电机停止"""
        self.serial.write(bytes([0x03]))
        response = self.serial.read(1)
        return response == b'\x55'

    def motor_brake(self):
        """AB电机刹车"""
        self.serial.write(bytes([0x04]))
        response = self.serial.read(1)
        return response == b'\x55'

    def heartbeat(self):
        """发送心跳包"""
        self.serial.write(bytes([0x00]))
```

## 7. 注意事项

1. **速度设置**：每次发送电机控制指令前，建议先设置速度
2. **停止vs刹车**：停止指令让电机惯性滑行，刹车指令立即抱死
3. **心跳机制**：必须定期发送心跳包(间隔≤100ms)以维持连接
4. **串口参数**：波特率115200，8N1格式
5. **通道选择**：0x01-0x04控制AB电机，0x05-0x08控制CD电机
